# Delivery Cost Calculator

Створити додаток для підрахунку вартості доставки замовлень у службі кур’єрської доставки. Основна логіка зберігається у функції з назвою calculate_delivery_cost. На вхід вона отримує відстань типу float, вагу посилки типу float та тарифну таблицю типу dict, та повертає число типу float, яке є вартістю доставки.
