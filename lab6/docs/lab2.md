# Лабораторна робота №2

**Тема:** Менеджери залежностей

**Завдання:**

1. Створити та налаштувати віртуальне оточення Python
2. Встановити бібліотеку requsts, продемонструвати її роботу
3. Створити файл з залежностями requirements.txt
4. Виконати перевірку вразливостей
5. Закомітити зміни
6. Встановити бібліотек axios, продемонструвати роботу
7. Виконати перевірку вразливостей
8. Закомітити зміни

## Запуск програми

```
cd lab2/
```

**Python**

```
cd Python/
./venv/Scripts/activate
pip install requests
python req_text.py
deactivate
```

**Node.js**

```
cd Node.js
npm install axios
node index.js
```
