Лабораторна робота 4

Тема: Автоматизація роботи з кодом

Завдання:

  1. Створити додаток на Python. Створити додаток що буде емулювати роботу сервісу бронювання готелів. Потрібно створити функції addRoom, bookRoom та calculateStayCost. Додаток повинний визначати вартість проживання залежно від кількості днів.
  2. Створити віртуальне оточення
  3. Встановити лінтер та форматер
  4. Згенерувати файл requirements.txt
  5. Відформатувати код згідно з вимогами лінтеру та форматеру
  6. Додати до .gitignore всі файли в папці з додатком, що не відносяться до коду.

Запуск програми

cd lab4/
./venv/Scripts/activate
pip install flake8 black pylint
python app.py
flake8 app.py
black app.py
pylint app.py
deactivate
