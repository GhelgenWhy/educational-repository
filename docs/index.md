**Тема:** Контроль версій

**Завдання:**

1. Зареєструвати обліковий запис на Github.
2. Завантажити та налаштувати Git.
3. Створити репозиторій.
4. Виконати перший коміт.

### Запуск програми

```
python hello_world.py
```

## Лабораторна робота 3

**Тема:** Системи збирання проєктів

**Завдання:**

1. Створити проєкт на C++. Створити додаток, що буде емулювати систему бронювання готелю. У файлах Room.h та Room.cpp міститься клас номера. У файлах Hotel.h та Hotel.cpp буде реалізація готелю. Файл main.cpp бронює номер та обчислює вартість проживання.
2. Створити Makefile, в якому будуть команди для компіляції та очищення проєкту.
3. Створити Dockerfile, який буде складатись з 2 етапів – компіляція додатку за допомогою утиліти make та його запуск.
4. Створити Docker-compose, який буде запускати додаток.

### Запуск програми

```
cd lab3/
docker-compose up
```

## Лабораторна робота 4

**Тема:** Автоматизація роботи з кодом

**Завдання:**

1. Створити додаток на Python. Створити додаток що буде емулювати роботу сервісу бронювання готелів. Потрібно створити функції addRoom, bookRoom та calculateStayCost. Додаток повинний визначати вартість проживання залежно від кількості днів.
2. Створити віртуальне оточення
3. Встановити лінтер та форматер
4. Згенерувати файл requirements.txt
5. Відформатувати код згідно з вимогами лінтеру та форматеру
6. Додати до .gitignore всі файли в папці з додатком, що не відносяться до коду.

### Запуск програми

```
cd lab4/
./venv/Scripts/activate
pip install flake8 black pylint
python app.py
flake8 app.py
black app.py
pylint app.py
deactivate
```

## Лабораторна робота 5

**Тема:** Перевірка правильності коду

**Завдання:**

1. Створити додаток на Python. Створити додаток який буде визначати запізнення працівника у робочій сфері. У ньому функція calculate_delay_minutes буде відповідати за обчислення хвилин запізнення, а is_late за перевірку факту запізнення. Головна функція буде використовувати ці функції для аналізу приходу працівника на роботу.
2. Покрити додаток тестами (unit-tests та doctests. )

### Запуск програми

```
cd lab5/
python L5_7.py
```
